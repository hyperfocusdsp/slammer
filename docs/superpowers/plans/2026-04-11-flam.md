# Flam, Ruff & Roll Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add per-step flam/ruff/roll (2/3/4 hits with global spread + humanize) to slammer's 16-step sequencer, wired through a pending-hit ring inside `KickEngine` so multi-stroke timing is sample-accurate across buffer boundaries.

**Architecture:** UI writes per-step state into `Sequencer::flam_state` (AtomicU8×16). On each step-boundary in `Plugin::process`, if the step is active, the plugin calls `engine.schedule_group(n_hits, spread_samples, humanize, params, velocity)`. The engine pushes N entries into its internal pending-hit ring with per-hit sample offsets and velocities (applying humanize via a local xorshift32 PRNG). The per-sample inner loop of `engine.process` ticks the ring and fires hits when their countdown reaches zero. Voice pool expands from 2 to 4 so roll tails don't steal each other.

**Tech Stack:** Rust, nih-plug, nih_plug_egui, parking_lot, serde, cargo test. RT-safe constraints (no heap allocs on audio thread except via `permit_alloc`).

**Spec:** `docs/superpowers/specs/2026-04-11-flam-design.md`

```rust
pub fn flam_state(&self, idx: usize) -> u8 {
    self.flam_state[idx].load(Ordering::Relaxed)

/// UI-thread only: advance the per-step flam state through Off → Flam →
/// Ruff → Roll → Off. No-op if the step is currently inactive, since a
/// flam on a rest is meaningless.
pub fn cycle_flam_state(&self, idx: usize) {
    if !self.is_step_on(idx) {
        return;
    }
    let next = (self.flam_state[idx].load(Ordering::Relaxed) + 1) & 0b11;
    self.flam_state[idx].store(next, Ordering::Relaxed);
}
```

```rust
pub fn set_step(&self, idx: usize, on: bool) {
    self.steps[idx].store(on, Ordering::Relaxed);
    if !on {
        self.flam_state[idx].store(0, Ordering::Relaxed);
    }
    let mut bits = self.persist_mirror.lock();
    if on {
        *bits |= 1u16 << idx;
    } else {
        *bits &= !(1u16 << idx);
    }
}
```

```rust
/// Push a hit into the first free ring slot. Silently drops the hit if the
/// ring is full (pathological — 12 slots covers 4 hits × 3 overlapping
/// steps, which is beyond musically reasonable).
fn push_pending(&mut self, samples_until: u32, velocity: f32) {
    for slot in &mut self.pending {
        if !slot.live {
            slot.samples_until = samples_until;
            slot.velocity = velocity;
            slot.live = true;
            return;
        }
    }
}

/// Advance the ring by one sample. Returns the velocities of any hits
/// that fired this sample. Writes go to a small stack buffer so no heap
/// allocation happens on the audio thread.
fn tick_pending_internal(&mut self, out: &mut [f32; PENDING_RING_SIZE]) -> usize {
    let mut count = 0;
    for slot in &mut self.pending {
        if !slot.live {
            continue;
        }
        if slot.samples_until == 0 {
            out[count] = slot.velocity;
            count += 1;
            slot.live = false;
        } else {
            slot.samples_until -= 1;
        }
    }
    count
}

#[cfg(test)]
pub fn tick_pending_for_test(&mut self) -> Vec<f32> {
    let mut out = [0.0f32; PENDING_RING_SIZE];
    let n = self.tick_pending_internal(&mut out);
    out[..n].to_vec()
}

#[cfg(test)]

#[cfg(test)]

}

/// xorshift32 — Marsaglia's 2003 variant. Produces a non-zero u32 as long
/// as the seed is non-zero. Fast, no branches, no allocation.
fn rng_xorshift32(&mut self) -> u32 {
    let mut x = self.rng_state;
    x ^= x << 13;
    x ^= x >> 17;
    x ^= x << 5;
    self.rng_state = x;
    x
}

/// Returns a centered, symmetric f32 in [-1.0, 1.0) derived from the PRNG.
/// Used for humanize jitter scaling.
fn rng_centered(&mut self) -> f32 {
    let u = self.rng_xorshift32();
    // Map u32 → [-1, 1) as (u as f32 / 2^31) - 1.0
    (u as f32 / 2_147_483_648.0) - 1.0
}
```

Add to `impl KickEngine`:

```rust
/// Schedule a group of 1–4 hits into the pending ring with sample-accurate
/// offsets relative to "now" (offset 0 fires on the next `tick_pending`).
///
/// * `n_hits` — 1, 2, 3, or 4. Any other value is clamped to 1.
/// * `gap_samples` — base inter-stroke gap in samples. Ignored when `n_hits == 1`.
/// * `humanize` — 0.0 (exact) to 1.0 (full jitter). Scales both timing
///   jitter (±20%×humanize) and velocity jitter (±10%×humanize).
/// * `step_velocity` — per-step velocity multiplier applied on top of the
///   per-hit base amplitude.
pub fn schedule_group(
    &mut self,
    n_hits: usize,
    gap_samples: u32,
    humanize: f32,
    step_velocity: f32,
) {
    let n = n_hits.clamp(1, 4);
    let base_vels: &[f32] = match n {
        1 => &[1.0],
        2 => &[0.7, 1.0],
        3 => &[0.7, 0.85, 1.0],
        4 => &[0.6, 0.75, 0.85, 1.0],
        _ => unreachable!(),
    };

    let humanize = humanize.clamp(0.0, 1.0);
    for (i, &bv) in base_vels.iter().enumerate() {
        // Base timing: first hit at offset 0, rest at i*gap.
        let base_offset = (i as u32).saturating_mul(gap_samples);

        // Timing jitter: ±20% of gap, scaled by humanize, skipped when
        // humanize == 0 (preserves the exactness contract).
        let offset = if humanize > 0.0 && gap_samples > 0 && i > 0 {
            let j = self.rng_centered() * 0.2 * humanize * gap_samples as f32;
            let signed = base_offset as i64 + j as i64;
            signed.max(0) as u32
        } else {
            base_offset
        };

        // Velocity jitter: ±10% of base, scaled by humanize.
        let vel = if humanize > 0.0 {
            let j = self.rng_centered() * 0.1 * humanize;
            (bv * step_velocity * (1.0 + j)).clamp(0.0, 2.0)
        } else {
            bv * step_velocity
        };

        self.push_pending_internal(offset, vel);
    }
}
```

Note the `i > 0` guard on timing jitter: the first hit stays at offset 0 so the group still "starts" on the step boundary. Without this, a flam could jitter-skew the whole group slightly late.
